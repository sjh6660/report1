### 论文 1：《ImageNet Classification with Deep Convolutional Neural Networks》

- 研究问题：解决大规模图像分类任务中，传统机器学习方法准确率不足、无法处理高维图像数据的问题；同时突破深层神经网络训练困难、易过拟合、计算效率低的瓶颈。

- 方法概述：

  设计了一个包含 5 个卷积层、3 个全连接层的深度卷积神经网络（AlexNet）；
  使用 ReLU 作为激活函数，解决了 Sigmoid 在深层网络中梯度消失的问题；
  采用双 GPU 并行训练，将模型拆分到两个 GPU 上，大幅提升训练速度；
  引入 Dropout 正则化，随机失活部分神经元，减少模型过拟合；
  使用重叠池化（Overlapping Pooling），提升特征提取的鲁棒性。

- 创新点：

  首次将深度 CNN 成功应用于大规模 ImageNet 图像分类任务；
  证明了 ReLU 激活函数在深层网络中的优越性；
  提出并验证了 Dropout 正则化在防止过拟合上的效果；
  实现了基于双 GPU 的模型并行训练，大幅提升了大模型训练效率。

- 实验设置：

  数据集：ImageNet 2010/2012（包含 120 万张训练图像、1000 个类别）； ◦ 硬件：两块 NVIDIA GTX 580 GPU； ◦ 训练：随机梯度下降（SGD），动量 0.9，权重衰减 5e-4，batch size 128； ◦ 评估指标：Top-1 错误率、Top-5 错误率。

- 主要结论：

  深度卷积神经网络可以在大规模图像分类任务上取得远超传统方法的效果；
  ReLU 激活函数能有效解决深层网络的梯度消失问题，加速模型收敛；
  Dropout 和数据增强能显著降低模型过拟合，提升泛化能力；
  GPU 并行训练是训练大规模深度神经网络的有效方式。

- 我没有理解的地方：

  双 GPU 模型并行的具体数据流动和同步机制；
  Dropout 在推理阶段如何与训练阶段的权重缩放配合；
  重叠池化相比普通池化，在理论上提升模型效果的底层原理。

- 这篇论文是否影响了我的兴趣判断：这篇论文让我直观感受到了深度学习在 CV 领域的突破性效果，理解了深度 CNN 的核心思想和关键技术，坚定了我继续学习 CV 方向深度学习的兴趣。

  ### 论文 2：《Deep Residual Learning for Image Recognition》

  - 研究问题：解决网络深度增加带来的退化问题：层数变深，训练误差不降反升，准确率饱和下降。 • 解决深层神经网络难以优化的问题，让极深网络能够轻松训练并持续涨点。 • 证明网络深度对视觉任务至关重要，并让训练超深网络成为可能。

  - 方法概述：

    残差学习：让层拟合残差函数 F (x) = H (x) - x，最终输出 H (x)=F (x)+x。
    捷径连接（Shortcut）：跨层直连，不增加参数与计算量，实现恒等映射。
    残差块设计：
    浅层：两层 3×3 卷积。
    深层：瓶颈结构（1×1 + 3×3 + 1×1），减少计算量。
    维度匹配：通道数不一致时用 1×1 卷积做投影映射。
    训练策略：BatchNorm、数据增强、SGD + 动量，不使用 Dropout。

  - 创新点：

    提出残差学习范式，从根本解决深度网络退化。 2. 无参数捷径连接，不增加计算与参数量，即插即用。 3. 成功训练极深网络（152 层），复杂度低于 VGG，精度更高。 4. 通用且易复现，成为后续几乎所有 CNN 与 Transformer 的基础结构。

  - 实验设置：

    数据集：ImageNet ILSVRC2015、CIFAR-10。 • 网络：ResNet-18/34/50/101/152。 • 优化器：SGD，batch=256，lr=0.1，动量 = 0.9，权重衰减 = 0.0001。 • 预处理：随机裁剪、水平翻转、减均值、BatchNorm。 • 评估：Top-1 错误率、Top-5 错误率、检测 mAP。

  - 主要结论：

    残差学习能轻松训练极深网络，解决退化问题。 2. 网络越深，性能越好，精度随深度持续提升。 3. 捷径连接的恒等映射无额外开销、极易实现。 4. 残差网络泛化性极强，可用于分类、检测、分割等所有 CV 任务。 5. 残差结构是现代深度学习模型的基础组件。

  - 我没有理解的地方：

    为什么学习 F (x) 比直接学习 H (x) 更容易优化。
    梯度在捷径连接中为什么能无损回传，解决梯度消失。
    瓶颈结构（bottleneck）的具体计算与通道变化逻辑。

  - 这篇论文是否影响了我的兴趣判断：

    ResNet 让我明白深度与残差连接是模型变强的关键，坚定了我继续学习CV、深度学习模型设计、深层网络训练的方向。

  ### 论文 3：《Long Short-Term Memory 》

  - 研究问题：

    解决传统循环神经网络（RNN）在处理长序列时的梯度消失问题，使模型能够有效学习并保留序列中的长期依赖信息，提升长序列建模的能力。

  - 方法概述：

    设计了 LSTM 单元，核心由 ** 细胞状态（cell state）** 和三个门控组成：遗忘门、输入门、输出门； 2. 遗忘门：控制上一时刻的细胞状态信息保留多少； 3. 输入门：控制当前时刻新信息加入细胞状态的程度； 4. 输出门：控制细胞状态信息对当前隐藏状态的输出； 5. 细胞状态作为 “记忆传送带”，只进行少量线性操作，让梯度在长序列中稳定传递，避免消失。   •

  - 创新点：

    首次提出了门控机制来控制循环网络中的信息流动；
    引入细胞状态，解决了 RNN 的梯度消失问题；
    让循环网络具备了长期记忆能力，为长序列建模提供了可行方案。

  - 实验设置：

    任务：人工长序列记忆任务、简单序列预测任务；
    对比模型：传统 RNN；
    评估指标：任务完成率、梯度范数变化、长期依赖建模效果。

  - 主要结论：

    LSTM 可以有效解决 RNN 的梯度消失问题，稳定训练长序列；
    门控机制让模型能选择性地记住和遗忘信息，更适合处理复杂序列；
    细胞状态的设计是 LSTM 能保留长期依赖的关键；
    LSTM 在长序列任务上的表现远超传统 RNN。

    

  - 我没有理解的地方：

    门控机制在反向传播时的梯度传递细节；
    细胞状态中 “恒定误差流” 的数学原理；
    不同初始化方式对 LSTM 训练稳定性的影响。

  - 这篇论文是否影响了我的兴趣判断：

    这篇论文让我清晰理解了 LSTM 的核心设计思想，也明白了序列建模中梯度消失问题的本质，坚定了我继续学习序列模型和循环神经网络的兴趣。
